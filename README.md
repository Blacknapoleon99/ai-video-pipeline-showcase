# AI Video Pipeline Showcase

Public-safe portfolio version of a AI-assisted video generation template. This repository preserves the useful application architecture while removing private history, secrets, credentials, customer/user data, local configuration, generated outputs, and private or copyrighted media.

## What This Demonstrates

TypeScript, Node.js scripts, Remotion, JSON episode templates, mock TTS/story workflows

## Safety Notes

This showcase intentionally excludes real .env files, production service identifiers, API keys, private media, generated databases, local machine paths, and original git history. Integration code is kept as architecture, but values are placeholders and must be replaced with your own test credentials for local development.

## Local Setup

Install dependencies with the package manager used by the project, then copy .env.example to your local env file and fill in only test/demo values.

```bash
npm install
npm run build
```

Some integrations are intentionally mock/demo-only in this public version. Use fake data and licensed assets only.

## Portfolio Context

This repo is meant for recruiter and engineering review: it shows project structure, UI flows, data models, integration boundaries, and implementation style without exposing private production material.
